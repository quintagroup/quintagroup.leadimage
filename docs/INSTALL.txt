quintagroup.leadimage Installation
----------------------------------

To install quintagroup.leadimage using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``quintagroup.leadimage`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        quintagroup.leadimage
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
